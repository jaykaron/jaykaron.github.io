Chroma

How To Play:

To land on a platform you must be the same color as it. Go as far as you can!

Controls:

f / d : Change Color

Space : Jump

p : Pause/Resume

Story:

This game was originally a Java applet made for a final project in high-school by Ari Karon in 2014. As 2014 passed and things progressed, Java applets became nearly impossible to run without installing a Java IDE. So I decided to remake Chroma with JavaScript. The game uses the PaperJS library for dealing with the canvas.
