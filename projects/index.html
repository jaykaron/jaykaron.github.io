This Website

After learning a little bit of HTML, CSS and JavaScript in high-school and university, I wanted to hone my skills and get some experience using some newer web technologies.

I started out using only vanilla jQuery and SASS, but after spending a considerable amount of time working on this cool nav-bar (please excuse the color scheme), I asked myself "Was that really worth it?" and switched to Bootstrap.

With Bootstrap it takes about 2 minutes to make the nav-bar at the top of this page which works across browsers and has mobile support. So that definitely sped things up.

After that it was pretty smooth sailing, all that was required was time. Time for going through Bootstrap's documentation, time for fiddling with the CSS, etc.

Takeaway

While I did familiarize myself with current tools used in making static websites. I realized, more importantly, that I don't particularly enjoy making static websites. There is little analyzing or heavy thinking involved, just finding the right part and fiddling around until it looks right.

Wikipedia Graph

A joint attempt with my brother to graph the links of Wikipedia and find out something usesful.

So far we have found out that their are lots of links. We are working on sorting them out.

We are using Python to scrape Wikipedia and putting the results into a Neo4J database, a system for graph databases that has a special query language called Cypher.

You can see the existing code in the Github repository. Here's a picture of all the articles connected to Science and the links between them to keep you busy.
